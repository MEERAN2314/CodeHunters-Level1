Display the Multiplication Table of a Number (Till 16 Steps)

Explanation:

Write a program to display the multiplication table of a given number up to 16 steps.
For example, for the number 5, the table should display 5 × 1 = 5, 5 × 2 = 10, ..., up to 5 × 16 = 80.

Example 1:

Input: n = 5
Output:
5 × 1 = 5
5 × 2 = 10
5 × 3 = 15
...
5 × 16 = 80


Example 2:

Input: n = 3
Output:
3 × 1 = 3
3 × 2 = 6
3 × 3 = 9
...
3 × 16 = 48
